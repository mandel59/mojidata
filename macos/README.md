# macOS: 選択文字 → IDS（mojidata）

`mojidata --select=ids` の結果から `IDS` だけを取り出し、候補が複数ある場合はカンマ区切りで出力して、キーボードショートカットで「選択中の漢字をIDS表記に置換」できるようにします。

## 使うスクリプト

- `macos/mojidata-ids.zsh`  
  入力（引数 or stdin）を `mojidata --select=char,ids` にかけて、`IDS` のみを `,` 区切りで標準出力します（`source` は無視）。
- `macos/ids-find-whole.zsh`  
  入力（引数 or stdin）を `ids-find --whole=...` にかけて、候補の漢字を `,` 区切りで標準出力します（複数候補を見やすくするため）。

例:

```sh
/bin/zsh macos/mojidata-ids.zsh --text 将
```

## Quick Action（サービス）の作り方（置換）

1. **Automator** を開く → **クイックアクション** を新規作成
2. 上部の設定:
   - 「ワークフローが受け取る現在の入力」: **テキスト**
   - 「対象」: **すべてのアプリケーション**
   - 「出力が選択されたテキストを置き換える」: **オン**
3. アクション **「AppleScriptを実行」** を追加
4. `macos/mojidata-ids-replace-selection.applescript` の内容を貼り付け
5. `repoPath`（デフォルトは `~/ws/mojidata`）を自分のパスに合わせて修正
6. 名前を付けて保存（例: `Mojidata: IDSに変換`）

## Quick Action（サービス）の作り方（合成: IDS → 漢字候補）

1. **Automator** を開く → **クイックアクション** を新規作成
2. 上部の設定:
   - 「ワークフローが受け取る現在の入力」: **テキスト**
   - 「対象」: **すべてのアプリケーション**
   - 「出力が選択されたテキストを置き換える」: **オン**
3. アクション **「AppleScriptを実行」** を追加
4. `macos/ids-find-whole-replace-selection.applescript` の内容を貼り付け
5. `repoPath`（デフォルトは `~/ws/mojidata`）を自分のパスに合わせて修正
6. 名前を付けて保存（例: `Mojidata: IDSから合成`）

### キーボードショートカット

**システム設定 → キーボード → キーボードショートカット → サービス** から、作成したクイックアクションにショートカットを割り当てます。

### 権限について

上記の設定（「テキストを受け取る」「出力で置換」）で作る場合、通常はキーボード操作の送信（UIスクリプティング）が不要なので **アクセシビリティ許可なし** で動きます。

もし「System Events でキー操作が許可されない」というエラーが出る作り方（クリップボード経由・キー送信）にしている場合は、そちらをやめてこの手順に寄せるのが確実です。
